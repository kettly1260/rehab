# 训练助手

训练助手是一个面向运动训练和康复训练的轻量级 PWA 网页应用。它可以在手机浏览器或安装为桌面应用后使用，支持动作计划编排、语音播报、计时训练、历史记录、主题颜色设置以及 S3/WebDAV 云端备份。

## 功能特性

- 训练计时：支持动作时长、次数、组数、次间休息、组间休息和项目间休息。
- 双侧训练：可为左右侧交替训练加入换边提示。
- 语音播报：训练开始、动作切换、次数提示、休息提示和倒计时播报。
- 语速调节：TTS 播报语速支持 `0.5x` 到 `3x`。
- 训练控制：开始、暂停、停止按钮悬浮在页面中下部，便于单手操作。
- 跳过阶段：可跳过当前播报或计时阶段。
- 历史记录：训练时长达到 20 秒后才会记录，避免误触产生无效记录。
- 方案库：可保存当前训练计划并再次载入。
- PWA 支持：可安装到手机桌面，提供基础离线缓存。
- 返回键保护：训练进行中按系统返回键不会直接退出页面，防止训练被误中断。
- 主题设置：支持默认蓝色、预设颜色、莫奈风格动态取色和自定义颜色。
- 云端同步：支持 S3 协议和 WebDAV 备份/恢复。

## 使用方式

### 添加训练动作

在训练页填写动作参数：

- 动作名称：例如深蹲、肩关节外旋、踝泵。
- 组数：该动作执行几组。
- 次数：每组执行几次。
- 单次秒数：每次动作保持或执行的时间。
- 次休秒数：同一组内每次之间的休息时间。
- 组休秒数：同一动作不同组之间的休息时间。
- 项休秒数：不同动作之间的休息时间。
- 双侧交替训练：启用后会按左侧、右侧顺序训练，并提示换边。

点击“添加到计划”后，动作会加入当前训练列表。

### 开始训练

点击页面中下部的播放按钮开始训练。训练开始后：

- 播放按钮变为暂停按钮。
- 停止按钮出现。
- 当前阶段、剩余时间、总用时、组数和次数会实时更新。
- 语音会按训练流程播报。

点击暂停按钮可暂停计时和语音，点击停止按钮可结束训练。

### 历史记录规则

训练总用时低于 20 秒时不会写入历史记录，也不会触发自动云备份。应用会提示“训练时间低于20秒，无法记录历史”。

训练总用时达到 20 秒后，结束训练会写入历史记录，并在已配置同步时自动备份到云端。

## 设置

底部导航中的“设置”页包含主题和同步配置。

### 主题颜色

支持以下主题方式：

- 默认蓝色主题。
- 紫色、绿色、橙色、玫红预设主题。
- 莫奈风格动态取色。
- 自定义颜色选择器。

主题配置会保存在本地，下次打开自动恢复。

### S3 同步

选择 `S3 协议` 后填写：

- Endpoint URL：S3 兼容服务地址。
- Region：例如 `us-east-1`。
- Bucket：存储桶名称。
- Access Key：访问密钥。
- Secret Key：密钥。

备份文件名固定为：

```text
rehab_pro_data.json
```

S3 服务需要允许浏览器跨域访问。若备份或恢复失败，请检查 S3 CORS 配置。

### WebDAV 同步

选择 `WebDAV` 后填写：

- WebDAV 地址：服务器目录地址。
- 用户名：WebDAV 用户名。
- 密码 / App Password：WebDAV 密码或应用密码。
- 备份文件路径：默认 `training_assistant_data.json`。

WebDAV 服务同样需要支持浏览器跨域请求，否则网页端无法直接访问。

### 备份内容

云端备份会保存完整本地数据，包括：

- 当前动作列表 `actions`
- 方案库 `routines`
- 训练历史 `history`
- 播报语速 `rate`

## 后台播放说明

应用已经启用以下 Web 能力以改善后台播放体验：

- Silent audio 保活。
- Screen Wake Lock。
- Media Session。
- 页面可见性恢复时自动续播。
- TTS watchdog 定期尝试恢复 `speechSynthesis`。
- PWA 返回键保护，避免误按返回导致页面卸载。

但移动系统和浏览器可能会限制黑屏或后台网页执行。尤其在省电模式、锁屏、系统清理后台或部分 iOS/Android 浏览器中，网页语音仍可能被暂停。这是系统策略限制，无法完全由网页绕过。

建议训练时：

- 将应用安装为 PWA 后使用。
- 训练中使用 Home 键进入后台，不要使用返回键关闭页面。
- 关闭系统对浏览器或 PWA 的过度省电限制。
- 需要稳定播报时尽量保持屏幕亮起。

## 本地运行

本项目是纯静态网页，不需要构建工具。可以用任意静态服务器运行。

使用 Python：

```bash
python -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```

使用 Node.js：

```bash
npx serve .
```

PWA 和 Service Worker 需要在 `localhost` 或 HTTPS 环境下才能正常工作。

## 文件结构

```text
index.html      页面结构
styles.css      Material 3 Expressive 风格样式
theme.js        主题颜色和莫奈动态取色逻辑
data.js         本地数据、动作列表、方案库和历史记录
workout.js      训练流程、计时、语音、后台保活和返回键保护
sync.js         S3/WebDAV 同步与自动备份
sw.js           PWA 离线缓存
manifest.json   PWA 应用配置
README.md       项目说明
```

## 数据与安全

所有训练数据和同步配置默认保存在浏览器 `localStorage`。如果填写了 S3 或 WebDAV 密钥，它们也会保存在本地浏览器中。

请不要在公共设备上保存同步密钥。不要把真实密钥写入仓库。

## 浏览器兼容性

建议使用最新版 Chrome、Edge 或 Android WebView。Safari 和部分国产浏览器可能对 PWA、后台播放、Wake Lock、SpeechSynthesis 或跨域请求有额外限制。
